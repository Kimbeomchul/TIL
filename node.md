# Node.js

### V8
- node js 는 V8엔진을 사용, V8엔진이란 크롬에서 사용하는 엔진.

### 특징
- 브라우저 밖에서 자바스크립트 코드를 실행가능.
- 이벤트기반 비동기 I/O 프레임워크
- 노드는 파일형태로 모듈을 관리할 수 있는 CommonJs로 구현
  + 기본모듈
  + 써드파티 모듈
  + 사용자 정의 모듈

### ExpressJS
- [익스프레스](http://expressjs.com/)는 Nodejs에서 대표적인 웹 프레임워크다. 아래 다섯가지 개념을 이해해야 제대로 사용할 수 있다. 
  + 어플리케이션
  + 미들웨어
  + 라우팅
  + 요청객체
  + 응답객체
